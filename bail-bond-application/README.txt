Bail Bond Application (English) Folder
=====================================

This folder contains the English version of the bail bond application form.

Contents:
- bail-bond-application.html: Full bail bond application form in English with integrated styling and form validation

Subfolders:
- scripts/: JavaScript files for this page
  * gas-script.js: Google Apps Script for handling form submissions and storing data in Google Sheets
- assets/: Images, videos, and other media files for this page (currently empty)
