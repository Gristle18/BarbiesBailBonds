Barbie's Bail Bonds Website
============================

This is the main repository for the Barbie's Bail Bonds website project.

Project Structure:
- HomePage/: Main homepage with company information and services
- OnlineApplication/: Language selection page for online applications  
- bail-bond-application/: English bail bond application form
- bail-bond-application-spanish/: Spanish bail bond application form
- docs/: Project documentation and setup files

Each page folder contains:
- HTML file for the page
- scripts/ subfolder for JavaScript files
- assets/ subfolder for images, videos, and other media
- README.txt describing the folder contents
