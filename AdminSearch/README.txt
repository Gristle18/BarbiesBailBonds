Admin Search Page
=================

Purpose:
- Administrative search interface for Barbie's Bail Bonds staff to search through uploaded documents and client data.

Structure:
- `index.html`: Page markup and styles.
- `scripts/`: Client-side logic and Google Apps Script integration.
- `assets/`: Placeholder for images or other media.
- `instructions.txt`: Setup and configuration instructions.

Notes:
- Phone number `561-247-0018` is rendered with `white-space: nowrap` to avoid line breaks (repo standard).
- External links (none by default) must use `target="_blank"`. Internal links should open in the same tab.
