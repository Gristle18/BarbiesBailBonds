HOW TO MAKE ALL TEXT SELECTABLE (INCLUDING U521981590 AND ALL POWER NUMBERS)
================================================================================

IMPORTANT: The PDF has been enhanced so ALL power numbers are now dark and clear.
           Adobe Acrobat's OCR will now be able to detect them all.

FILE TO USE: 09152025_FINAL_PROPER_SIZE.pdf

STEP-BY-STEP INSTRUCTIONS FOR ADOBE ACROBAT:
--------------------------------------------

1. OPEN THE PDF
   - Open Adobe Acrobat (NOT just Reader - you need full Acrobat)
   - File → Open → Select "09152025_FINAL_PROPER_SIZE.pdf"

2. ACCESS OCR TOOLS
   - Click "Tools" in the top menu
   - Find and click "Scan & OCR"

3. START TEXT RECOGNITION
   - Click "Recognize Text" button
   - Select "In This File"

4. CONFIGURE SETTINGS (IMPORTANT!)
   - Click the Settings gear icon
   - Set these options:
     • OCR Language: English (US)
     • PDF Output: "Searchable Image" (BEST OPTION - keeps enhanced quality)
     • Downsample to: 300 dpi
   - Click OK

5. RUN OCR
   - Click "Recognize Text" button
   - Wait for processing (progress bar will show)
   - This will take 2-5 minutes for 51 pages

6. SAVE THE OCR VERSION
   - File → Save As
   - Name it: "09152025_FINAL_WITH_TEXT.pdf"

TROUBLESHOOTING:
================================================================================

IF TEXT ISN'T SELECTABLE:

- Ensure you selected "Searchable Image" not "Editable Text"
